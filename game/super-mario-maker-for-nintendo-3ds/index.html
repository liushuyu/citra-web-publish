Super Mario Maker for Nintendo 3DS

Summary

Super Mario Maker for Nintendo 3DS will run perfectly in Citra under certain conditions. The game is able to progress past the start menu while booting for the first time, but most times after a restart, it will stop working and will hang on the screen saying ‘Accessing SD Card…’ where Yamamura won’t be able to finish the last piece. Unless you dump extdata of this game from your 3DS and import it to Citra, it won’t reach the title screen.

Workarounds

Delete the current savedata “Progress” (and extdata; done automatically by the game) then restart. Make a save state for each mode? (Maker, Mario Challenge)
