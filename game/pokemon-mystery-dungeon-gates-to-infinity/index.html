Summary

Pokémon Mystery Dungeon: Gates to Infinity suffers from minor graphical issues, but is otherwise playable from start to finish. Crashes should not be expected. The game should be expected to run near full speed on modest hardware inside dungeons, but slowdowns should be expected in Paradise and other overworld environments.
