Summary

Metroid: Samus Returns has some slight problems in Citra. Graphically, running the game at certain higher internal resolutions causes texture errors to occur on the map. This is the only major/obvious graphical issue. Audio is mostly fine with a crackle here and there, but this game does require more modern hardware to attain full speed. The game is fully playable from start to finish and attains almost perfect emulation, despite these very minor issues.
